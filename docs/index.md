# Bedrock Health Group Governance & Compliance Policy Library

This is the controlled governance and compliance policy library for Bedrock Health Group.

This library is intended to support structured policy management, document control, review workflow, version history, and future policy access.

## Library Purpose

The purpose of this library is to provide a centralized location for approved policies, templates, and policy governance materials.

## Current Status

This library is maintained through Bedrock Health Group's controlled GitHub review and publication workflow. Policies listed in the Policy Index reflect the current public policy library records.

## Library Areas

- Compliance Program
- Healthcare Compliance
- Privacy & Security
- Governance
- Operations
- Finance & Administration
- Human Resources
- Vendor Management
- Training & Acknowledgements

## Document Control Principle

Policies should be drafted, reviewed, approved, published, revised, and retired through a controlled process.
